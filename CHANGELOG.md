# Changelog

All notable changes follow [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and semantic versioning.

## [Unreleased]

- Added source-built analysis support and hosted-runner verification for macOS ARM64 and Windows x86-64.

## [1.0.0] - 2026-07-13

### Added

- Linux `strace` recording with descendants, bounded output, atomic storage, and early redaction.
- Rust parsing, normalization, process modeling, seven deterministic diagnoses, evidence-based ranking, and completeness penalties.
- Terminal, JSON, local visual, portable bundle, and offline HTML workflows.
- Six reproducible Linux demos, schemas, fixtures, CI, release packaging, installer, and documentation.

### Security

- Added bounded trace parsing, core diagnostics, event serialization, bundle extraction, and comparison loading.
- Replaced shell archive import with a path-safe Rust extractor that rejects links, duplicates, unexpected entries, traversal, and oversized data.
- Minimized persisted environment data, applied private storage permissions, hardened report headers, and disabled framework telemetry.
- Added locked dependency audits, immutable workflow actions, SBOM generation, release provenance attestations, deterministic archives, and clean-install verification.
